# Question Images

Supported formats: `.png`, `.jpg`, `.jpeg`, `.webp`.

## Question 1

One shared image for hair color:

```text
assets/questions/q1/hair.png
```

You can use `.jpg`, `.jpeg`, or `.webp` instead of `.png`.

## Question 2

Image depends on the answer from question 1:

```text
assets/questions/q2-appearance-by-hair/brunette.png
assets/questions/q2-appearance-by-hair/blonde.png
assets/questions/q2-appearance-by-hair/red.png
assets/questions/q2-appearance-by-hair/brown.png
```

Optional fallback:

```text
assets/questions/q2-appearance-by-hair/default.png
```

## Question 3

One shared image for body type:

```text
assets/questions/q3/body.png
```

## Optional Later Images

Question 4 and 5 can also show images if you add:

```text
assets/questions/q4/image.png
assets/questions/q5/image.png
```
